# MedAnalyzer — Система анализа медицинских данных

**MedAnalyzer** — веб-приложение для автоматизированного анализа медицинских файлов и ведения персонального дневника самочувствия. Система предназначена для пациентов и помогает отслеживать состояние здоровья между визитами к врачу.

> **Важно**: Результаты анализа носят **информационный характер** и **не являются диагнозом**. Диагноз может быть поставлен только лечащим врачом.

---

## Функционал

###  Анализ медицинских файлов
- Поддержка форматов: `.dcm`, `.pdf`, `.docx`
- Максимальный размер файла: **50 МБ**
- Анализ изображений с помощью **предобученной модели ResNet50**
- Анализ лабораторных PDF через **экспертную систему на основе клинических норм**
- Время обработки: ≤30 сек (изображения), ≤60 сек (документы)

### Дневник самочувствия
- Оценка самочувствия по 5-балльной шкале (с визуализацией смайликами)
- Выбор симптомов из списка: головная боль, лихорадка, слабость и др.
- Возможность ввода **пользовательского симптома**
- Привязка записей к дате (нельзя выбрать будущую дату)
- Полный CRUD: создание, просмотр, редактирование, удаление

### Безопасность
- Аутентификация через сессии (SessionAuthentication)
- Защита от CSRF, XSS, SQL-инъекций
- Обработка персональных данных

### Интерфейс
- Адаптивный дизайн (мобильные + десктоп)
- Цветовая палитра: синий, голубой, белый
- Единый стиль для всех модулей
